Un deuxième paragraphe

La fonction: createElement(), permet de créer un élément CSS comme parexemple un élémnet p (comme paragraphe)

La fonction: createTexteNode() permet d'insérer du texte

La fonction : appendChild() permet d'ajouter le nouveau élément comme par exemple p ou le h1 aprés le dernier élément du body

La fonction : createTextNode permet d'ajouter du texte dans un élément

La fonction insertBefore() permet d'insérer un élément entre deux autres éléments du CSS. Exemple insertBefore (L'élément à insérer, l'élément avant lequel il faut l'insérer)
